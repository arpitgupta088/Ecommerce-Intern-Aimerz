# Ecommerce-Intern-Aimerz
This is a full-Stack E-Commerce Web Application for internship, built with MERN (MongoDB, Express, React, Node.js). Includes secure user authentication, product catalogue, shopping cart, and checkout functionality with responsive UI.

## Frontend Overview

The frontend of this electronics e-commerce website is built using React. It provides a modern, responsive user interface optimized for desktop and mobile devices. Key features include:

- **Product Catalogue:** Browse and search electronic products with filtering and sorting options.
- **Product Details:** View detailed information, images, and specifications for each item.
- **Shopping Cart:** Add, update, or remove products from the cart with real-time price calculations.
- **User Authentication:** Secure login and registration using JWT.
- **Checkout Process:** Seamless checkout flow with order summary and payment integration.
- **Order History:** Users can view their past orders and track current orders.
- **Responsive Design:** Ensures a smooth experience across all devices.
